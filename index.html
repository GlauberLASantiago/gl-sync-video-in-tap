Este app foi criado para facilitar o trabalho de criação musical no MuseScore, pois este atualmente não tem a possibilidade de se carregar um vídeo e sincronizá-lo. A ideia é que o estudante de trilha sonora abra um arquivo do MuseScore e, quando ele clica no teclado, o GL Sync-in-Tap literalmente ouve o som e inicia o vídeo junto. Além disso, o app possui diversas funcionalidades, como de importar e exportar arquivos MusicXML que podem ser lidos pelo MuseScore, possibilitando diversas facilidades no processo de pontuar locais no vídeo para se fazer trilha sonora.

Como usar:

  1. Carregar Vídeo: Selecione um arquivo local ou use um link do YouTube ou URL direta no painel "Arquivo".
  2. Marcar Sincronismo:
    • Use + Pontual (Atalho: M) para batidas ou momentos únicos.
    • Use + Contínua (Atalho: C) para trechos com duração (clique uma vez para iniciar e outra para parar).
  3. Som das Marcas: Reproduz cliques nas pontuais e notas musicais diatônicas sobrepostas (a partir de C4, E4, G4, Bb4) nas marcas contínuas.
  4. Edição: Um clique simples na marca vai para o início dela (2º clique vai para o fim da contínua; 3º clique desmarca). Clique duplo permite editar o nome e a contagem musical.
  5. Exportar: Baixe em formato MusicXML para abrir no MuseScore, organizando as marcas contínuas em pautas separadas ("Contínuas 1" a "Contínuas 3") de acordo com as notas atribuídas.
